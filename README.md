# CoinTalk - ChatRoom
코인톡 백엔드 서버 채팅방 도메인
코인방별로 채팅 하는 기능입니다.

<br><br>
## 배경(Background)
코인 차트를 보며 같이 토론을 하고, 차트 캔버스에 그림을 그려서 저장 후 채팅방에서 공유하기 위해 필요한 채팅 기능입니다.

<br><br>
## 목표(Goals)

 - 코인별 단체 채팅방 구현
 - 채팅방 입장 시 이전 대화 내용 불러오기

<br><br>
## 목표가 아닌 것 (Non-goals)
 - 일대일 채팅
<br><br>
## 계획 (Plan)

 - Webflux를 이용한 Secket 통신 구현
 - 코인방 별로 분리
